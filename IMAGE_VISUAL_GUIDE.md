## CSS Styling Applied to Images

### Logo Image
```css
.logo-image {
    height: 50px;
    width: auto;
    object-fit: contain;
}
```

### Certification Logos (NEW)
```css
.cert-logo {
    width: 100%;
    height: 180px;
    object-fit: contain;
    padding: 15px;
    background: linear-gradient(135deg, rgba(255, 107, 107, 0.05), 
                                        rgba(0, 217, 255, 0.05));
    border-radius: 10px;
    margin-bottom: 15px;
}
```

### Profile Image
```css
.about-image img {
    width: 100%;
    border-radius: 15px;
    box-shadow: 0 10px 40px rgba(255, 107, 107, 0.2);
    border: 3px solid #ff6b6b;
    transition: all 0.35s cubic-bezier(0.4, 0, 0.2, 1);
}

.about-image img:hover {
    transform: scale(1.05);
    box-shadow: 0 15px 50px rgba(255, 107, 107, 0.3);
}
```

### Program Card Images
```css
.program-card img {
    width: 100%;
    height: 250px;
    object-fit: cover;
    transition: all 0.35s cubic-bezier(0.4, 0, 0.2, 1);
}

.program-card:hover img {
    transform: scale(1.08);
}
```
